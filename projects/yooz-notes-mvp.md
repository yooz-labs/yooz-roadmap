# Yooz Notes MVP

**Status:** MVP Complete, Preparing Public Beta
**Target:** Q4 2025 Public Launch
**Platform:** Web, iOS, Android (via Capacitor)

## Overview

Privacy-first note-taking app with local AI capabilities and end-to-end encryption.

## Key Features

### Core Functionality
- [x] Markdown editor with live preview
- [x] Mermaid diagram support
- [x] Full RTL language support (Arabic, Hebrew, Farsi)
- [x] CJK language support (Chinese, Japanese, Korean)
- [x] File organization and folders
- [x] Search functionality

### Privacy & Sync
- [x] End-to-end encryption
- [x] Local-first storage
- [ ] Encrypted sync (In Progress)
- [ ] Export/Import functionality
- [ ] Backup options

### AI Features
- [ ] Local AI summarization (December 2025)
- [ ] Local AI translation (December 2025)
- [ ] Smart text completion
- [ ] Privacy-preserving learning

### Platform Support
